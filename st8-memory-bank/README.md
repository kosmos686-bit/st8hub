# ST8-AI Memory Bank

Единый источник правды для всех AI-агентов ST8-AI.

## Структура

```
memory-bank/
├── productContext.md    — Компания, продукты, стек, партнёры
├── systemPatterns.md    — Стандарты, правила, паттерны, реквизиты
├── clientPipeline.md    — Пайплайн клиентов, статусы, контакты
├── activeContext.md     — Что делается прямо сейчас, открытые вопросы
├── decisionLog.md       — Архитектурные и бизнес-решения (почему так)
└── progress.md          — Сделано / в работе / следующие шаги
```

## Установка

### Шаг 1 — Агенты

```bash
# Скопировать агентов в Claude Code
cp -r st8-agents/* ~/.claude/agents/

# Или для Cursor
cp st8-agents/*.md .cursor/rules/
```

### Шаг 2 — Memory Bank

```bash
# Скопировать memory-bank в корень рабочего проекта
cp -r memory-bank/ ~/st8-workspace/memory-bank/
```

### Шаг 3 — Запуск

```bash
cd ~/st8-workspace
claude  # запустить Claude Code
```

Первая команда:
```
Активируй ST8 AI Director. Прочитай memory-bank/ и доложи текущий статус.
```

## Команды

```
"Активируй ST8 AI Director — [задача]"
"Активируй ST8 KP Architect — КП для Lucky Group"
"Активируй ST8 Security Auditor — проверь этот код"
"UMB"  →  обновить Memory Bank из текущей сессии
```

## Обновление Memory Bank

После значимых событий (новый клиент, новое решение, завершённый этап):

```
UMB  →  агент сам обновит нужные файлы
```

Или вручную — отредактировать нужный .md файл.
