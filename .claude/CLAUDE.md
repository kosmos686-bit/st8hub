# ST8 AI — Режим экономии токенов

## Критично: не жри контекст

### Запрещено
- Читать файлы >50 строк целиком без разрешения
- View без диапазона строк
- Переписывать файл целиком ради одной строки
- Менять кавычки/отступы в незатронутых файлах
- Делать Read всего проекта при поиске одной ошибки
- Писать длинные вступления перед правкой

### Обязательно
1. Поиск первый — Grep/findstr перед View
2. Фрагменты — View только с диапазоном (View 45-65)
3. Точечные правки — Edit только сломанной строки
4. Batch — правки списком за один вызов
5. Regex — замена паттерна через Replace
6. Compact — /compact каждые 20 сообщений

### Алгоритм при ошибке
Grep "текст ошибки" → View строки ±10 → Edit → Проверить

### Работа с логами
- Не читай лог целиком. tail -20 или последние 30 строк.
- Grep по trace, не читай весь файл.

### Ответы
- Максимум 2 предложения объяснения.
- Дальше — только команды (Grep/View/Edit/Bash).
- Если всё понятно — молчи и правь.
